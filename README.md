# Climate-Math-Research

**Course:** Climate Mathematics — San Diego State University  
**Language:** Python (Jupyter Notebook)  
**Grade:** A  
**Type:** Group project

Collaborative project analyzing PM2.5 pollution patterns from the 2017 Thomas Fire. The notebook (`Wildfire_PM2.5_Thomas2017.ipynb`) processes and visualizes air quality data to assess the spatial and temporal spread of particule matter following the wildfire event. A written report is also included (`Genty_Wildfire_PM2.5_Thomas2017.pdf`).

**Authors:** Olympe Genty,  Timmy Liang, Lauren Scott — San Diego State University
